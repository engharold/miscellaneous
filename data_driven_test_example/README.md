Exemplo de teste automatizado usando Python + Selenium Webdriver. 

O script interage com várias abas no navegador, usa os conceitos Data-Driven Testing (teste dirigido a dados) e Web Scraping (coleta de dados da web) para gerar a massa de dados de forma dinâmica, e no final, exibe o resultado do teste para cada registro. 

O script pode fazer o cadastro de várias pessoas; só precisamos enviar o argumento quando executamos o script desde a linha de comando. O vídeo mostra um exemplo do script sendo executado, onde é testado o cadastro de duas pessoas.
