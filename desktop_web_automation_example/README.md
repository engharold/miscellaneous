Exemplo de automação simultânea de aplicações desktop e web, usando Python + Selenium Webdriver + Lackey. 

Para que o script funcione corretamente, precisa-se instalar o Selenium Webdriver (automação web) e o Lackey (automação desktop), no ambiente virtual de Python. O script deve ser executado com privilégios de administrador para que não dê erro a parte de automação de desktop.
